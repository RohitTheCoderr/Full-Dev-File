Semantic element

HTML Semantic Elements are elements that clearly describe their meaning to both the browser and the developer. They are used to create a structure for the web page, making it easier to understand and maintain. Examples of semantic elements include <header>, <footer>, <nav>, <article>, <section>, and <aside>1
Some more information about semantic elements in HTML is that they have intrinsic meaning and convey that meaning both to the browser and the developer.
